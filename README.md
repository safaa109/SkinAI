# SkinAI

SkinAI is a simple web application for skin disease classification using a fine-tuned ResNet-18 model.

## Features

* Upload a skin image
* Predict skin disease using CNN (PyTorch)
* Simple Flask web interface

## How to Run

1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
